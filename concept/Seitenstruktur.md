# Optimale Seitenstruktur (Navigation)

## 1. Startseite

**Zweck:** Schneller Überblick für alle.

Inhalte:

* Kurzbeschreibung der Wanderung auf dem Malerweg
* Zeitraum der Tour
* Teilnehmerliste
* wichtigste Eckdaten (Gesamtlänge, Etappen, Höhenmeter)
* Übersichtskarte
* Schnellzugriffe auf:

  * Heute / aktuelle Etappe
  * Etappenübersicht
  * Unterkünfte
  * Navigation

**Navigationseinstieg für alle Inhalte.**

---

# Hauptnavigation

## 2. Etappen

**Wichtigster Bereich der Seite**

### Struktur

```
Etappen
 ├── Übersicht
 ├── Etappe 1
 ├── Etappe 2
 ├── Etappe 3
 ├── ...
 └── Etappe N
```

### Etappen-Übersicht

* Karte der gesamten Route
* Liste aller Etappen
* Basisdaten:

  * Start / Ziel
  * Kilometer
  * Höhenmeter
  * Gehzeit

### Etappenseite (Template)

```
Etappe X: Start → Ziel

1. Basisdaten
2. Karte & GPX
3. Highlights
4. Pausenmöglichkeiten
5. Schwierige Stellen
6. Alternativen / Abkürzungen
```

Typische Highlights entlang des Weges:

* Bastei
* Schrammsteine
* Kuhstall

---

# 3. Unterkünfte

Struktur:

```
Unterkünfte
 ├── Übersicht (Karte)
 ├── Unterkunft 1
 ├── Unterkunft 2
 └── ...
```

Unterkunftsseite:

* Name
* Adresse
* Check-in
* Frühstück
* Kontakt
* Buchungsstatus
* Entfernung zur Route

Optional:

* Einkaufsmöglichkeiten
* Restauranttipps

---

# 4. Organisation

Zentrale Gruppeninfos.

```
Organisation
 ├── Tagesplan
 ├── Teilnehmer
 └── Treffpunkte
```

Inhalte:

**Tagesplan**

* Startzeiten
* Frühstückszeiten
* geplante Pausen
* Abendessen

**Teilnehmer**

* Liste
* Kontaktmöglichkeiten

**Treffpunkte**

* Startpunkt
* wichtige Zwischenpunkte

---

# 5. Navigation

Sehr wichtig **für unterwegs auf dem Smartphone**.

```
Navigation
 ├── GPX-Tracks
 ├── Karten
 └── Apps
```

Inhalte:

* GPX Downloads
* Offlinekarten
* Links zu:

  * Komoot
  * Outdooractive

Optional:

* QR-Codes für Tracks

---

# 6. Anreise

```
Anreise & Abreise
 ├── Anreise
 ├── Parken
 └── Rückreise
```

Typische Orte:

* Pirna
* Bad Schandau

Inhalte:

* Bahnverbindungen
* Parkmöglichkeiten
* Rücktransport

---

# 7. Wetter

Kurzbereich mit:

* Wettervorhersage
* Sonnenaufgang / Sonnenuntergang
* aktuelle Wegbedingungen

Relevant für den Nationalpark Sächsische Schweiz.

---

# 8. Downloads

Zentrale Sammlung:

* GPX-Tracks
* Karten
* Packliste
* Zeitplan
* Teilnehmerliste
